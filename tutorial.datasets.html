Using the OSM Data Manager

We also provide an OSM Data Manager which, in addition to using the default cache (~/.cache/geowrangler/osm), also caches the geodataframe for either the pois or the roads datasets from OSM in memory to avoid having to reload the OSM data from disk.

# Create the osm data manager
odm = geofabrik.OsmDataManager()
pois_ph = odm.load_pois("philippines")
